About LearnSphere

LearnSphere is a multi‑modal learning assistant for Machine Learning. It generates explanations, code examples, audio lessons, and visual learning aids, powered by Google Gemini. Content is personalized to your learning level (beginner, intermediate, or advanced).

What you can do

  • Get ML concept explanations at different depth levels (text streams as it’s generated).
  • Generate runnable code (e.g. Python, Java) and download it; code streams as it’s generated.
  • Create audio lessons (script + MP3 via gTTS) tailored to your level.
  • Generate ML concept diagrams and flowcharts with Gemini image generation.
  • View and manage your activity in History (delete items or clear all).
  • Optionally enable experimental RAG with scikit-learn docs to ground text answers.

Privacy & storage

Generated files (code, audio, images) are stored locally in the storage/ directory on the server you run. Your API key is stored only in your browser’s localStorage when you save it in Settings; the server does not store your key and uses it only for the request at hand.

Built with

Flask, Google Gemini (text and image), gTTS. Optional RAG over scikit-learn docs (experimental).
